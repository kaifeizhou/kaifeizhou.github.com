webpack插件

插件可以帮我们在打包之前或者打包之后做些事情，常用的插件有如下几个：

html-webpack-plugin

这个插件可以帮我们在打包后自动在output对应的文件夹中创建一个html文件，并将打包生成的js文件引入到页面上，如果希望按照我们的模板来创建html，可以在src文件夹中创建一个html模板，首先安装这个插件：

npm i html-webpack-plugin -D

然后在webpack.config.js中做如下配置：

const HtmlWebpackPlugin = require('html-webpack-plugin');

module:{
    ......
},
plugins:[ new HtmlWebpackPlugin({ template:'src/index.html' })]

其中，”src/index.html“，是src提供的html模板，打包时会按照这个模板生成html文件，如果不提供模板，打包会按照默认的方式生成html文件。

clean-webpack-plugin

这个插件可以在下载打包之前，清空public文件中的文件，让public文件夹中的文件没有多余遗留的文件，首先安装这个插件：

npm i clean-webpack-plugin -D

然后在webpack.config.js中做如下配置：

const { CleanWebpackPlugin } = require('clean-webpack-plugin');

module:{
    ......
},
plugins:[ 
    new HtmlWebpackPlugin({ template:'src/index.html' }),
    new CleanWebpackPlugin()    
    ]

HotModuleReplacementPlugin

这个插件可以在我们修改样式时，更新页面中标签的样式效果而不用刷新整个页面，这个需要配合webpack开发服务器一起使用。这个插件属于webpack上的插件，所以不用另外安装了，这个插件简称为HMR，配置如下：

const webpack = require('webpack');
plugins:[
    new HtmlWebpackPlugin({
        template:'src/index.html'
    }),
    new CleanWebpackPlugin(),
    new webpack.HotModuleReplacementPlugin()
]
