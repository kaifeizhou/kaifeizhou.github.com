开发服务器

webpack提供了一个在开发过程中使用的服务器模块，开启之后，我们修改项目代码后，项目会自动打包，而且显示页面会自动刷新显示最新的效果，这个给我们开发带来了很大的便利。首先安装这个模块：

npm i webpack-dev-server -D

接着在webpack.config.js中配置相关参数：

devServer:{
    contentBase:'./public',
    open:true,
    port:8080,
    hot:true
}

其中，contentBase设置服务器根目录，open:true 可以让我们在开启服务器时，会自动打开一个浏览器窗口来运行我们的项目，port设置服务器运行的端口，hot:true表示让浏览器开启HotModuleReplacementPlugin这个功能。

接下来我们在package.json文件中的scripts中配置运行这个服务器模块：

"scripts": {
    "dev": "webpack",
    "start": "webpack-dev-server"
  },

然后，我们就可以用如下命令开启服务：

npm run start
