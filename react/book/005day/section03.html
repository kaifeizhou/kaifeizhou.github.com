安装第三方包

官方脚手架里面已经提供了常用的包，在这基础上，针对项目开发的需求，还需要安装以下这些包：

axios

无论是在vue项目的开发，还是在react项目的开发中，目前还是首选axios作为ajax程序编写的的包。 安装方式：

yarn add axios

react-router-dom

在react项目中使用路由，需要用到这个包 安装方式：

yarn add react-router-dom

redux

在react项目将数据集中管理，方便组件之间传递数据，就要使用这个包 安装方式：

yarn add redux

react-redux

redux的优化扩展包 安装方式：

yarn add react-redux

swiper

移动端制作轮播图的首选js库 安装方式：

yarn add swiper

echarts-for-react

react中插入echart图表，需要安装两个包 安装方式：

yarn add echarts echarts-for-react

react-pullload

react中制作下拉刷新，上拉加载更多的功能的包 安装方式：

yarn add react-pullload
