WebSocket

WebSocket 是 HTML5 开始提供的一种在单个 TCP 连接上进行全双工通讯的协议。

传统的HTTP是半双工模式，就是客户端可以发请求找服务端要数据，而服务端不能给客户端推送数据，这种方式对于需要实时更新的数据就比较麻烦，传统解决方法是使用轮询的方式，通过定时器，间隔一段时间给浏览器发请求来要数据。

WebSocket使用的是“ws”协议，它属于全双工模式，就是说客户端可以向服务端拉取数据，服务端也可以向客户端推送数据，在 WebSocket API 中，浏览器和服务器只需要完成一次握手，两者之间就直接可以创建持久性的连接，并进行双向数据传输。所以WebSocket特别适合应用在在线聊天，直播等场景中。

创建WebSocket应用

websocket应用分为服务端和客户端，首先，服务端需要创建一个websocket服务，我们以nodejs作为服务端开发语言，那么创建服务可以使用"ws"模块，下面创建的是一个接受客户端信息并且向客户端广播的程序：

const WebSocket = require('ws');
const wss = new WebSocket.Server({port:7891});

wss.on('connection', function connection(ws) {
    ws.on('message', function incoming(data) {
        wss.clients.forEach(function each(client) {
            if (client.readyState === WebSocket.OPEN) {
              client.send(data);
            }
        });
    });

    ws.send('你好！欢迎登录聊天系统')
});

接下来创建客户端程序，客户端的接口不多，就下面几个：

// 创建一个websocket连接：
let ws = new WebSocket('ws://127.0.0.1:7891');

// 接收服务端推送过来的消息
ws.onmessage = (ev)=>{
   console.log(ev.data);             
}

// 给服务端发消息
ws.send('消息')
