reacjs开发

fetch

fetch号称是AJAX的替代品，是在ES6出现的，使用了ES6中的promise对象。Fetch是基于promise设计的。Fetch的代码结构比起ajax简单多了，参数有点像jQuery ajax。但是，fetch不是ajax的进一步封装，而是原生js，没有使用XMLHttpRequest对象。

fetch发送get请求

fetch('http://localhost:7890/slide')
.then(function(res) {
    return res.json();
})
.then(function(myJson) {
    console.log(myJson);
}).catch(function(err){
    console.log(err)
})

其中，第一个then是对返回的值做处理，如果是需要想得到json，就使用res.json()方法，如果想得到字符串，就使用res.text()方法。第二个then得到的是最终结果。get方法传递参数可以将参数写在地址栏后面，类似："?aa=12&bb=36"的方式。

fetch发送post请求

fetch('http://localhost:7890/login', {
    body: JSON.stringify(params), 
    method: 'POST',
    headers: new Headers({
        'Content-Type': 'application/json'
    })
})
.then(function(res) {
    return res.json();
})
.then(function(myJson) {
    console.log(myJson);
}).catch(function(err){
    console.log(err)
})
